Install & setup apache web server on 5 Sep 2019

The set-up server follow from the web site https://www.ostechnix.com/install-apache-mysql-php-lamp-stack-on-ubuntu-18-04-lts/

Install apache

DO NOT TYPE ANYTHING AFTER "#" TAG!!

  1. sudo bash #need to be root user

  2. #update and upgrade the system before install new package
    apt update && apt upgrade -y

  3. # to install apache run
    apt install apache2 -y

  4. # config for the mod rewrite
    vi /etc/apache2/apache2.conf

    # press :wq to save and exit from edit the config file to restart the apache service type

    systemctl restart apache2
